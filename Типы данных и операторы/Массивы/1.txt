/**
 * Практика #1
 * Что произойдет в результате выполнения следующего кода?
 * @author Aleksandra
 */

В строках
   int a1[] = {1,2,3,4,5};
   int a2[] = {6,7,8,9,10};
создаются два массива целых чисел: a1 и a2.

a1 содержит значения {1, 2, 3, 4, 5}.
a2 содержит значения {6, 7, 8, 9, 10}.

Далее в строке
   a1 = a2;
происходит переназначение, при котором ссылка переменной a1 теперь указывает на массив, на который указывает a2.

Это значит, что:
Массив, который изначально был в a1 (то есть {1, 2, 3, 4, 5}), больше недоступен через a1, и теперь a1 ссылается на массив {6, 7, 8, 9, 10}.
Массив, находящийся в a2, теперь доступен через обе переменные. То есть оба a1 и a2 указывают на один и тот же массив ({6, 7, 8, 9, 10}).

Если больше нет ссылок на массив {1, 2, 3, 4, 5}, этот массив становится недоступным, и в результате может быть подвержен сборщику мусора (Garbage Collector), который освобождает память, занимаемую недоступными объектами.
