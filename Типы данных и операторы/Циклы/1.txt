/**
 * Практика #1
 * Приведите тривиальные примеры бесконечных циклов для конструкций while и do-while.
 * @author Aleksandra
 */

Пример бесконечного цикла с использованием конструкции while:

while (true) {
    System.out.println("Этот цикл никогда не завершится!");
}

Объяснение: В данном примере цикл while использует условие true, которое всегда истинно, поэтому выполнение цикла будет продолжаться бесконечно.


Пример бесконечного цикла с использованием конструкции do-while:

do {
    System.out.println("Этот цикл тоже никогда не завершится!");
} while (true);

Объяснение: Как и в предыдущем примере, цикл do-while также использует условие true.
