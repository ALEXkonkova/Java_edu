Какая область видимости у интерфейса, объявленного следующим образом:

interface Interface {
  ...
}

Ответ: По умолчанию интерфейс имеет пакетную область видимости, если не прописаны модификаторы доступа (public, protected, private).
В этом случае интерфейс может быть доступен только внутри того же пакета. Классы, находящиеся в других пакетах, не смогут реализовать или использовать этот интерфейс.
